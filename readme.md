# To Do List - React Bootstrap

Aplicación web para el control de tareas y metas personales.

## Actividad
Unidad I – React Bootstrap + CSS

## Tecnologías utilizadas
- React
- React Bootstrap
- Bootstrap
- CSS

## Descripción
Este proyecto corresponde a una maqueta HTML desarrollada en React.
No contiene lógica funcional, únicamente diseño, conforme a los requerimientos
de la actividad.

## Ejecución
1. Clonar el repositorio
2. Instalar dependencias con `npm install`
3. Ejecutar con `npm start`
